HTML

Introducción

HTML significa hypertext markup language/lenguaje de marcado de hipertextos.
Un hipertexto es un texto que contiene enlaces a otros textos.
No es un lenguaje de programación, sino un lenguaje basado en etiquetas, llamados lenguajes de marcado.
Para colocar un comentario en HTML se hace asi: 

<!-- COMENTARIO-->

Bases

Cuando comenzamos un archivo HTML debemos añadir la siguiente etiqueta: 

<!DOCTYPE html>

Como estarás viendo HTML utiliza etiquetas, que tienen la estructura de <nombre de etiqueta> y normalmente tienen etiqueta de cierre. Hasta ahora ninguna de las etiquetas mencionadas utiliza una de esta (recuerda que siempre que una etiqueta lleve una de cierre será visible en el bloque de código de ejemplo.)
Las etiquetas de cierre tiene la estructura de </nombre de etiqueta>.

Luego de utilizar la etiqueta de !DOCTYPE debemos usar las siguientes etiquetas: 

<html> 
 code 
</html>

Dentro de las etiquetas html van dos importantes, <head> y <body>, ambas con etiquetas de cierre.
Hasta ahora un archivo .html nos estaría quedando algo así: 

<!DOCTYPE html> 
<html lang="idioma"> 
<head>  
 code 
</head> 
<body>  
 code 
</body>
</html>

Seguramente te diste cuenta de que dentro de la etiqueta html coloque algo que todavía no se explicó, bien, eso es porque es el próximo tema, los atributos.

Atributos

Los atributos son información adicional en una etiqueta, sirven para identificar, personalizar o configurar el comportamiento o apariencia de una etiqueta.
Casi siempre tienen la siguiente estructura: nombre del atributo="valor" y se colocan dentro de la etiqueta de apertura luego del nombre de esta, además en una misma etiqueta puedes poner cuantos atributos necesites.
Entonces ¿qué es el atributo dentro de la etiqueta html? Es el atributo lang y sirve para indicar el idioma principal de la página.
Para ello usamos versiones acortadas de estos, llamados códigos de idiomas(definidos por la ISO 639-1), por ejemplo "es" para el español, "en" para el inglés, "ja" para el japonés, "de" para el alemán, etc.
Aquí tienes una página para encontrar más de estos. 

<html lang="es">
